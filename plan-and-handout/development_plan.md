# EBU6304 助教招聘系统开发计划

## 项目概览

- **项目名称**: 北京邮电大学国际学院助教招聘系统
- **技术栈**: Java Servlet/JSP Web 应用 (需要 Tomcat)
- **界面**: HTML + CSS + 浏览器访问
- **数据存储**: JSON 文件
- **团队规模**: 6 人

---

## Git 提交规范

### 提交信息格式:

```
<type>: <description>

[optional body]
```

### Type 类型:

- `feat`: 新功能
- `fix`: Bug修复
- `docs`: 文档
- `style`: 代码格式
- `refactor`: 重构
- `test`: 测试
- `perf`: 性能优化
- `chore`: 杂项

### 每人需有:

- 独立的功能分支
- 至少10次有意义的提交
- 合并到main的PR记录
